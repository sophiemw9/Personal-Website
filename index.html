Sophie's Work

CARP Project

For this task, I read through chapters 2-6 in “The Non-Designers Design Book” to develop my understanding in Contrast, Alignment, Repetition and Proximity within design. Using this knowledge, I had to create an A3 poster for each design principle using content provided.

Typography Project

For this task, I was given information to include about an exhibition in Bristol. Using this information, I needed to create 8 different posters each representing one of the 8 typographic systems. The challenge outlined that we could only use 1 of the available typefaces, the canvas must be 200mm x 200mm. In addition to black, I could use only 1 other colour and I may not include images/ illustration only shapes and lines.

Coding Worksheets

Each week, I was provided worksheets to work through to help develop my knowledge and understanding in HTML and CSS. I learnt the basics in setting up a website, to adding styles and design, to understanding flexbox. In addition, these worksheets gave me insight into using GitHub to host a website.
